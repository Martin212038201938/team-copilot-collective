# Setup Guide — Empfohlene kostenlose Accounts

## Bereits vorhanden

### Google Search Console
- URL: https://search.google.com/search-console
- Status: Alle drei Domains verifiziert
- Liefert: Indexierungsdaten, Crawling-Fehler, Klicks, Impressionen, Rankings

### Bing Webmaster Tools
- URL: https://www.bing.com/webmasters/
- Status: Alle drei Domains verifiziert
- Liefert: Bing-Indexierung, Crawling-Daten, Suchperformance

## Kein Account nötig

### Google PageSpeed Insights API
- URL: https://developers.google.com/speed/docs/insights/v5/get-started
- Die API ist ohne Registrierung nutzbar (allerdings mit Rate-Limit)
- Liefert: Core Web Vitals, Performance Score, Accessibility Score
- Tipp: Für höhere Limits kann ein kostenloser API-Key über die
  Google Cloud Console erstellt werden

## Optional empfohlen

### Google Analytics 4 (GA4)
- URL: https://analytics.google.com
- Warum: Traffic-Daten, Nutzerverhalten, Conversion-Tracking
- Aufwand: ~15 Min Setup pro Domain, Tracking-Code einbinden
- Vorteil fürs Dashboard: Echte Besucherzahlen neben den GSC-Impressionen

### Google Tag Manager
- URL: https://tagmanager.google.com
- Warum: Vereinfacht das Einbinden von Analytics und anderen Tags
- Aufwand: ~10 Min Setup, ein Container für alle Domains möglich

## Perspektivisch interessant

### Ahrefs Webmaster Tools (kostenlos)
- URL: https://ahrefs.com/webmaster-tools
- Warum: Backlink-Profil, organische Keywords, Site-Audit
- Einschränkung: Nur für verifizierte Domains, weniger Daten als Paid
- Aufwand: ~10 Min Setup pro Domain

### Screaming Frog SEO Spider (Free Version)
- URL: https://www.screamingfrog.co.uk/seo-spider/
- Warum: Tiefgehender Technical SEO Crawl (bis 500 URLs kostenlos)
- Vorteil: Findet Broken Links, Redirect-Ketten, Duplicate Content
- Einschränkung: Desktop-Tool, manuell ausführbar

### Yandex Webmaster (für internationale Sichtbarkeit)
- URL: https://webmaster.yandex.com
- Warum: Falls die Seiten auch in Osteuropa gefunden werden sollen
- Priorität: Niedrig für DACH-fokussierte Seiten
